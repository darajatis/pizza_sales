# Pizza Sales Analysis

## Overview

This project analyzes pizza sales data to uncover sales trends, customer purchasing behavior, and product performance insights.
The project is implemented in **two versions**, each producing nearly similar dashboards but using different analytics and visualization tools. This approach demonstrates flexibility in data workflows and the ability to deliver consistent insights using multiple technologies.

* **Version 1:** SQL Server (SSMS) + Excel
* **Version 2:** SQL Server (SSMS) + Power BI

The main objective is to showcase end-to-end data analytics skills, from database creation and SQL querying to data analysis and dashboard visualization.

---

## Dataset

The dataset contains transactional pizza sales data, including:

* Pizza and order id
* Pizza name
* Quantity
* Unit and total price
* Order date and time
* Pizza category and size
* Pizza ingredients

**Format:** Excel (.xlsx)
**Granularity:** Order-level sales data

---

## Project Workflow

### Version 1: SSMS + Excel

1. Imported raw Excel data into SQL Server
2. Created database and tables in SSMS
3. Wrote SQL queries to extract and aggregate sales data
4. Connected Excel to SQL Server
5. Performed data cleaning and analysis in Excel
6. Built a sales dashboard using Excel charts and pivot tables

### Version 2: SSMS + Power BI

1. Imported raw Excel data into SQL Server
2. Created database and tables in SSMS
3. Wrote SQL queries for business analysis
4. Connected Power BI to SQL Server
5. Processed and modeled data in Power BI
6. Built an interactive sales dashboard using Power BI visuals

---

## Dashboard

Both versions present nearly similar insights, including:

* Total sales and total orders
* Sales trends over time
* Sales by pizza category and size
* Best-selling pizzas
* Performance comparison across categories

The dashboards are designed to be clear, business-focused, and easy to interpret for stakeholders.

---

## Key Results & Insights

* Identified top-performing pizza categories and sizes
* Highlighted peak sales periods and ordering patterns
* Revealed revenue contribution by product type
* Demonstrated consistent insights across Excel and Power BI dashboards

## How to Run This Project

### 1. Set Up Database

* Open SQL Server Management Studio
* Import the Excel dataset into SQL Server
* Create the required database and tables
* Run the provided SQL queries

### 2. Run Version 1 (Excel Dashboard)

* Open the Excel file
* Connect Excel to SQL Server
* Refresh data connections
* Review analysis and dashboard sheets

### 3. Run Version 2 (Power BI Dashboard)

* Open the `.pbix` file in Power BI Desktop
* Connect to SQL Server
* Refresh the dataset
* Explore the interactive dashboard
